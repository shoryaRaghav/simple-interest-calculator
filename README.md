# Simple Interest Calculator

This project calculates **Simple Interest** using the formula:

**Simple Interest = (Principal × Rate × Time) / 100**

## Features

* Calculates simple interest quickly
* Easy to use
* Takes principal, rate of interest, and time as input

## Formula

SI = (P × R × T) / 100

Where:

* **P** = Principal amount
* **R** = Rate of interest (per year)
* **T** = Time (in years)

## Example

Input:

* Principal = 1000
* Rate = 5
* Time = 2

Output:

* Simple Interest = 100
